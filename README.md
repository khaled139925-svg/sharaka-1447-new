# منصة شراكة - الخدمات المتكاملة 🚀

هذا المشروع مبني باستخدام تقنية **React** و **Vite** و **Tailwind CSS**.

---

## 🛠️ كيف تشغل المنصة في 3 خطوات بسيطة:

إذا واجهت "صفحة فارغة" أو "شعلة نار"، فهذا يعني أنك تحاول فتح الملف مباشرة. اتبع هذه الخطوات في الـ Terminal داخل مجلد المشروع:

### 1️⃣ تثبيت المكتبات (مرة واحدة فقط)
```bash
npm install
```

### 2️⃣ تشغيل المنصة (للمعاينة والتطوير)
```bash
npm run dev
```
> سيظهر لك رابط مثل `http://localhost:5173` قم بفتحه في المتصفح.

### 3️⃣ بناء النسخة النهائية (للنشر الفعلي)
إذا أردت تشغيل الموقع بشكل نهائي ومستقر:
```bash
npm run build
npm run preview
```

---

## ✨ التعديلات الأخيرة المضافة:

✅ **حقل الصورة الشخصية/الشعار**: تم إضافة إمكانية رفع صورة أو شعار في شاشة "انضم إلينا" للمستشارين مع ميزة المعاينة والحذف.
✅ **إصلاحات Vite**: تم تحديث إعدادات `vite.config.ts` لضمان عمل الموقع في كافة البيئات (Local & Remote).
✅ **تحسينات الأداء**: تم تحسين عملية الـ Build لضمان استقرار الموقع.

---

## 📁 هيكل المشروع:
- `client/src/pages`: تحتوي على كافة صفحات الموقع (الرئيسية، انضم إلينا، إلخ).
- `client/src/components`: تحتوي على العناصر القابلة لإعادة الاستخدام.
- `server/`: يحتوي على كود الخادم وقاعدة البيانات (إذا لزم الأمر).
